# banking-app-education-tr

**Eğitim Amaçlı Mobil Bankacılık Uygulaması** — A single-page Turkish mobile banking education demo built with plain HTML, CSS, and JavaScript. No build step or dependencies required.

## Features

- **Genel Bakış (Dashboard)**: Displays account balance, monthly income/expense summary, and a live transaction history.
- **Transfer**: Submit money transfers with Turkish IBAN validation and balance checking. Accepted transfers are instantly reflected in the dashboard stats and transaction list.
- **Eğitim (Education)**: Six interactive cards covering key banking concepts (debit vs. credit cards, interest, security, mortgage, budgeting, and transfer methods).
- **Persistence**: All state (balance and transactions) is automatically saved to `localStorage` so data survives page refresh.

## Usage

Open `index.html` in any modern web browser — no installation or server needed.
